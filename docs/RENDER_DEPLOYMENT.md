# Render Deployment

Use this app as a Node web service.

## Build command

```bash
npm install && npm run build
```

## Start command

```bash
npm run start
```

## Required Environment Variables

```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=
NEXT_PUBLIC_APP_URL=
AUTH_SECRET=
```

Set `NEXT_PUBLIC_APP_URL` to your Render URL after deployment.
